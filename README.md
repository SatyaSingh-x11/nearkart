# NearKart 🚀

NearKart is a hyperlocal discovery platform that helps users find products available in nearby shops in real-time.

## 🔥 Features
- Real-time product search
- Nearby shop discovery
- Price + availability visibility
- Mobile-first UI/UX
- Progressive Web App (PWA)

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 Vision
To become India's default way to discover and buy products locally.
